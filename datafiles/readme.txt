==Controls==

Arrow Keys - Menu Navigation
Z - Accept
X - Cancel
C - Extra Menu Action
Z/X/C + Arrow Keys - Party Actions
Spacebar + Arrow Keys - Items (TBI)
LeftShift - Shift Menu
